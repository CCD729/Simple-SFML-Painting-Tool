# Simple-SFML-Painting-Tool
GAME 230 Project 1. A simple tool created with C++ and SFML that allows users to draw stuff in a window and save their work as a .png picture.

Thank you for trying this simple painting tool!

Hotkeys
===========
Left mouse button - Draw
Space - Switch Circle/Square brush
Arrow Left [←] /Right [→] - Change brush color hue (color)
Comma [,] / Period [.] - Change brush color saturation
Arrow Up [↑] / Down [↓] - Change brush color value (brightness)
Hyphen [-] / Equal [=] - Change brush size
L Ctrl+Z - Undo last stroke
L Ctrl+S - Save current canvas to png file (in the same folder with the painting tool)


感谢你试用这个简易画图器！

热键
===========
鼠标左键 - 绘画
空格键 - 切换圆形/正方形笔刷
左右方向键 [←] [→] - 调整笔刷色相（色调）
逗号与句号 [,] [.] - 调整笔刷饱和度
上下方向键 [↑] [↓] - 调整笔刷明度 （亮度）
连字符（减号）与等于号[-] [=] - 调整笔刷大小
左Ctrl键+Z - 撤销前一笔画
左Ctrl键+S - 保存当前画布内容至png文件（与本程序位于同一文件夹）
